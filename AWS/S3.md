# 基本
S3にデータを上げると3AZ(Availability Zone)以上
# AZ
AWSリージョン内の冗長電源、ネットワーク、設備を備えたデータセンターのこと
# バケットポリシー
バケットとその中のオブジェクトへのアクセス許可を付与できるリソースベースのポリシー
CloudFrontでホスティングするとビヘイビアで推奨ポリシーがコピーできる
20KBのサイズ制限がある
# アクセスコントロール(ACL)
他アカウントに基本的な読み取り・書き込みアクセス許可を付与するもの
# ライフサイクル設定
データのライフサイクル設定が出来るらしい
一定期間過ぎたら別リソースに保存or削除
毎日24時(UTC)に実行される
